Launchpad

Library to access a Novation Launchpad in a unix environment

Download .zip Download .tar.gz View on GitHub

launchpad

This library can be used to access a Novation Launchpad from a unix machine. To connect, a block-device is used.

Currently the library cannot be used, it will only show the data received by the launchpad.

usage

You can find a example receiving and sending data in BounceToLaunchpadExample.java in the test-directory.

To retrieve messages, implement a LaunchpadHandler and register it to the launchpad instance. The LaunchpadHandler will now be called always, when a Command is received.

    launchpad.addHandler(new LaunchpadHandler() {
        public void recieve(MidiCommand command) {
            System.out.println(command);
        }
    });

To send Commands, you can use the methods off(x, y) or set(x, y, color).

    launchpad.off(1, 2);
    launchpad.set(2, 3, Color.RED);
